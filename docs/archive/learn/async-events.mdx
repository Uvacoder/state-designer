# Async Events

You can wait on promises with async events.

```jsx
async: {
  await: () => fetch("https://dog.ceo/api/breeds/image/random"),
  onResolve: {
    do: (data, payload, result) => console.log("Got an image: " + result),
  },
  onReject: {
    do: (data, payload, result) => console.log("Had a problem!", result),
  }
}
```
